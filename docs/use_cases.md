1. Use Case — Government Scheme Discovery
Scenario

A citizen wants to find government schemes they qualify for.

User Input
I am a farmer in Tamil Nadu earning less than 2 lakh.
Which schemes can I apply for?
System Actions

1 Detect intent → scheme discovery
2 Ask eligibility questions
3 Check scheme database
4 Return eligible schemes

Expected Output
You may be eligible for the following schemes:

1 PM-KISAN
2 Pradhan Mantri Fasal Bima Yojana
3 State Farmer Subsidy Scheme

3. Use Case — Domestic Violence
Scenario

A victim asks for help.

User Input
My husband is beating me.
System Actions

1 Detect crisis category
2 Provide emergency helpline
3 Show legal protection information
4 Suggest support organizations

Expected Output
You may be experiencing domestic violence.

Immediate help:
Women Helpline: 1091
Emergency: 112

You have legal protection under the Protection of Women from Domestic Violence Act.
4. Use Case — Police Refusal to File FIR
Scenario

User complains about police refusal.

User Input
Police refused to register my FIR.
System Actions

1 Detect category → police refusal
2 Retrieve legal law
3 Provide escalation steps

Expected Output
Under CrPC Section 154, police must register an FIR.

If they refuse, you may:
1 Approach the Superintendent of Police
2 File a complaint to a Magistrate under CrPC 156(3)
